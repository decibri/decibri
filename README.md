# decibri

Cross-platform audio capture, output, and processing for Node.js and browsers.

Rust core with native addon (Node.js) and WASM (browser) bindings. One `npm install decibri` works everywhere.

## Status

Under development. See the [existing npm package](https://www.npmjs.com/package/decibri) for the current stable release.
