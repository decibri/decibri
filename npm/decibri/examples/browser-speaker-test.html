decibri browser audio playback test

Tap a button to play audio through the browser Speaker, then listen and watch the log. Playback must start from a tap (the browser only allows audio after a user gesture), so nothing plays until you tap. Serve this page over localhost or https; opening it as a file:// URL will not work. See examples/README.md for serve and mobile instructions.

Format:
about 1.5 s. Should be a clean, steady tone.
Listen for any gaps, clicks, or dropouts.
Forces resampling. Should match the first button's pitch.
Should halt immediately.
Idle.
